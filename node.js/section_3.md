##디버그

	node inspect debugging.js
	debug> list(10) ->10줄 디버그
	debug> n -> 다음라인으로 넘어감
	debug> c -> 코드에 debugger 지정된곳으로 이동
	debug> control+c control+c 종료
	debug> repl

-
	nodemon으로 실행 코드 변경시 디버깅이 재시작 된다
	nodemon inspect app.js read --title="hello"
